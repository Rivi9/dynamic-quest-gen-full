## Notes for Analysis

- Primary dependent variable: **Flow subscale** (H1 in the study)
- Secondary DV: **Immersion subscale** (H2)
- Report Cronbach's α for each subscale (target ≥ 0.70)
- Use independent-samples t-test: Experimental vs Control
- Report Cohen's *d* for effect size (target *d* ≥ 0.5)
